## 🚀 How to Run the Dashboard

Since this dashboard is built using standard, highly optimized **Vanilla HTML5, CSS3, and JavaScript**, you don't need complex frameworks, compilation steps, or an active internet connection to run it.

### Option 1: Direct File Open (Easiest)
1. Open your File Explorer and navigate to: `C:\Users\umang\Documents\ssc_cgl_dashboard\`
2. Double-click the [index.html](file:///C:/Users/umang/Documents/ssc_cgl_dashboard/index.html) file.
3. It will open instantly in your default web browser (Chrome, Edge, Firefox, Brave, etc.).

### Option 2: Live Local Server (Recommended for dynamic resizing)
If you are using VS Code or have Python installed:
- **Python**: Open a terminal in this directory and run:
  ```bash
  python -m http.server 8000
  ```
  Then open `http://localhost:8000` in your browser.
- **VS Code**: Install the "Live Server" extension, open the folder, and click **"Go Live"** in the bottom status bar.

## 🎯 Premium Features Built for Your Success

1. **Focus Mode (Today's Missions)**:
   - Placed front-and-center on the Dashboard page. It displays *only* the specific topics mapped to your current study day, preventing analysis paralysis.
   - Includes checkmarks for **Learned** (concepts clear), **Practiced** (50+ PYQs solved), and **Mastered** (solved in <45s).
   - Dynamic tracker shows your **Daily Goal Progress** based on active study topics and daily rituals.

2. **40-Day Structured Study Plan**:
   - Organized into **4 distinct study phases**:
     - **Phase 1: Foundations (Days 1–10)**: Focus on speed drills, basic vocabulary, parts of speech, and computer fundamentals.
     - **Phase 2: Applications (Days 11–25)**: Cover profit & loss, interest, time & work, active/passive voice, polity, and biology.
     - **Phase 3: Advanced (Days 26–35)**: Deep dive into Geometry, Mensuration, Algebra, Syllogisms, Cloze tests, and modern history.
     - **Phase 4: Simulations & revision (Days 36–40)**: Take intense daily full-length mocks, revise formulas, and test computer qualifying rules.
   - Click "Complete Today's Mission & Advance" to automatically roll the dashboard forward, update the day counter, and unlock tomorrow's syllabus target.

3. **Multiple Syllabus Views (Interactive Switcher)**:
   - **Tree Hierarchy**: An accordion list grouping subjects into categories, topics, and subtopics with checkmarks.
   - **Interactive Mind Map**: A custom SVG-driven canvas featuring smooth Bezier linkages, color-coded subject families, zoom-in/out, drag-to-pan, and clickable nodes that expand and highlight topics instantly.
   - **Table Layout**: A tabular directory of every single subtopic. Fully **searchable**, **filterable** (by subject, difficulty, weightage, and status), and features a **"Quick Master"** button.
   - **Grid Cards**: A clean deck of high-level subject cards displaying readiness percentage scores.

4. **Mock Test Analytics**:
   - Log your mock tests with date, score (out of 200), accuracy %, rank, sectional scores (Quant/Reasoning/English/GA), and review notes.
   - Renders a custom **interactive SVG line chart** showing score progression. Includes a green dashed target line at **140/200** (the safety cutoff indicator).

5. **Study Toolkit**:
   - **Mistake Book**: A central repository to save questions you got wrong, tricky rules, and formulas. Categorized and searchable.
   - **Pomodoro Focus Timer**: Built-in 25-minute study / 5-minute break timer with an animated SVG countdown ring. Features background **Synthesized Voice Speech Alerts** (it speaks to you when the session finishes!).
   - **Stopwatch**: A general session timer to record exactly how long you study.

6. **Full State Persistence (Offline Mode)**:
   - Everything you check off, mock scores you enter, day counters, mistakes logs, and Pomodoro presets are saved instantly in your browser's `localStorage`. You will never lose progress.

7. **⚡ Sectional Challenge Conquest Mode (15-Min Test)**:
   - Challenge yourself in any subject (Maths, English, Reasoning, or Computer) by triggering a **15-minute sectional run**.
   - You must answer **25 questions** sequentially. Tab-switching and custom controls are locked during the run to mimic exam conditions.
   - **Progressive Difficulty Scaling**: Questions start at **Easy** (Qs 1–5), advance to **Medium** (Qs 6–15), and peak at **Hard** (Qs 16–25) with dynamic math option shufflers and curated database pools.
   - **Passing Threshold**: Score a minimum of **20/25 (80%)** to clear the sectional cutoff and unlock "Super Human Reflexes" rating!
